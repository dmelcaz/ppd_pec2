Tarea 1.3

Publica el fichero .html y el vídeo en un servidor web para verla con una dirección http pública y vuelve a analizar sus características. Si no tienes ningún servidor web, puedes utilizar cualquier de los hostings gratuitos que existen en Internet.

El fichero .html junto con los vídeos se ha publicado en la siguiente dirección aprovechando la funcionalidad de hosting que proporciona GitHub: dirección donde está subido. Por otro lado, al servirse el vídeo como una petición HTTP GET al fichero correspondiente al clip este mantiene las características con las que fue codificado manteniendo el formato del contenedor, códecs y demás características.
